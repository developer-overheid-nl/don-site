Open source is de manier van werken waarbij de makers van systemen de broncode
openbaar beschikbaar maken met een opensource-licentie. Hierdoor kan iedereen de
software bekijken en hergebruiken. De overheid volgt het beleid ‘Open, tenzij’:
software van de overheid moet zoveel mogelijk open source zijn, behalve als er
goede redenen zijn om de software niet beschikbaar te maken.

## Belangrijke redenen

- **Transparantie** - Iedereen die dat wil, kan zien hoe de software werkt en in
  elkaar zit. Dit leidt tot vertrouwen in digitale diensten van de overheid.
- **Hergebruik** - Verschillende overheidsinstellingen, ondernemers en inwoners
  kunnen samenwerken aan software, en van elkaar leren. Ontwikkelaars kunnen
  gebruikmaken van elkaars ideeën, en samen sneller innoveren.
- **[Wet van Linus](https://nl.wikipedia.org/wiki/Wet_van_Linus)** - Iedereen
  kan fouten of risico’s ontdekken en onderzoeken. Dit vergroot de veiligheid en
  betrouwbaarheid van bijvoorbeeld apps en inlogmethodes.
- **Innovatie** - Doordat open software herbruikbaar is, kan open source werken
  innovatie en ontwikkelprocessen versnellen. Overheden kunnen bijvoorbeeld
  software van elkaar gebruiken. Dit voorkomt dubbel werk en onnodige kosten.
  Ook is de overheid niet afhankelijk van één softwareleverancier. Zo voorkom je
  "vendor lock-in".
- Woo en Who-compliant-by-default.
- Als je Open Source werkt is de voortgang van een project inzichtelijk, en dus
  ook de besteding van overheidsgelden.
