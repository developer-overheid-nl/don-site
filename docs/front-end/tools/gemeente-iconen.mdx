> **Duidelijke iconen helpen bezoekers van gemeentelijke websites om sneller bij de juiste informatie te komen. Daarom ontwikkelde OpenGemeenten deze iconenset, maak er gebruik van.**

Gemeenteiconen is een iconenset die bestaat uit 200+ iconen die speciaal zijn ontworpen voor gemeentelijke websites. De iconen zijn gratis te gebruiken en te downloaden.

[Naar gemeenteniconen.nl](https://www.gemeenteniconen.nl/iconen)

## Installatie en gebruik in React

Voor het gebruik van gemeenteiconen in een React project is er een handige tutorial beschikbaar. Hierin wordt uitgelegd hoe je de iconen kunt installeren en gebruiken in je project.

[Naar de tutorial](https://frameless.github.io/iconset-npm/)
