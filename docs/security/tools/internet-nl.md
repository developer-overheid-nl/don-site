Internet.nl is een online testtool van het Platform Internetstandaarden waarmee
je kunt controleren of een website, mailserver of internetverbinding voldoet aan
moderne internetstandaarden. De tools controlen onder andere op IPv6, DNSSEC,
HTTPS, HSTS, en beveiligde e-mailstandaarden zoals DMARC, DKIM en SPF.

De test geeft inzicht in welke standaarden correct zijn geïmplementeerd en toont
aanbevelingen voor verbeterpunten. Dit maakt het een praktisch hulpmiddel voor
overheidsorganisaties die willen voldoen aan de 'Pas toe leg uit'-lijst van het
Forum Standaardisatie.

**[Voer een check uit op internet.nl](https://internet.nl/test-site/)**
